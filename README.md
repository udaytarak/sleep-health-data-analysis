# Sleep Health and Lifestyle Data Analysis

This project explores how lifestyle factors influence sleep quality using Python data analysis tools.

## Project Objectives

The analysis answers the following questions:

- Does stress level affect sleep duration?
- Does physical activity improve sleep quality?
- Which occupations have poor sleep quality?
- Are sleep disorders related to lifestyle habits?
- Does age affect sleep patterns?
- Do people with higher daily steps sleep better?

## Tools Used

Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  
Google Colab  

## Project Workflow

1 Data Understanding  
2 Data Cleaning  
3 Exploratory Data Analysis  
4 Data Visualization  
5 Insights and Conclusions  

## Key Insights

- Higher stress levels are associated with shorter sleep duration
- Physical activity improves sleep quality
- Sleep disorders show higher stress levels
- Sleep duration strongly affects sleep quality
